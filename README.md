### 🔬 DSIG — Gestor de Semilleros de Investigación

![C#](https://img.shields.io/badge/C%23-239120?style=flat-square&logo=c-sharp&logoColor=white)
![SQL Server](https://img.shields.io/badge/SQL%20Server-CC2927?style=flat-square&logo=microsoft-sql-server&logoColor=white)

> Plataforma de gestión académica para semilleros de investigación con tres tipos de actores y acceso diferenciado por rol.

| Actor | Funcionalidades |
|-------|----------------|
| 👨‍💼 **Administrador** | Gestión global de usuarios, configuración de semilleros y generación de reportes |
| 👨‍🔬 **Líder de Semillero** | Creación de proyectos, asignación de tareas y seguimiento del equipo |
| 🧑‍💻 **Investigador** | Consulta de actividades, registro de avances y entrega de evidencias |

- ✅ Control de acceso basado en roles (RBAC) con permisos diferenciados por tipo de usuario
- ✅ Diseño orientado a Clean Code y separación clara de responsabilidades por capas
